# Trigger Deploy on Netlify - Quick Guide

## 3 Simple Steps

1. **Go to**: https://app.netlify.com/projects/mos-calculator2/overview

2. **Click**: **"Deploys"** tab (top menu)

3. **Click**: **"Trigger deploy"** â†’ **"Clear cache and deploy site"**

4. **Wait**: 2-3 minutes for build to complete

5. **Done!** Your site is updated.

---

## Why "Clear cache and deploy site"?

- Clears build cache
- Ensures environment variables are picked up
- Recommended after adding/updating environment variables

---

## What Happens Next?

1. Build starts automatically
2. You'll see build progress in real-time
3. Green checkmark when done
4. Your site is live with new changes
